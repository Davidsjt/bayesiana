El código fué probado en linux (Ubuntu)

Para ejecutar el código es necesario tener instalado "python3", 
	sudo apt install python3

Y las bibliotecas:
1. pandas
	sudo apt install python3-pandas
	
2. nlkt
	sudo apt install python3-nltk

3. sklearn
	sudo apt install python3-sklearn

Para ejecutar el código es necesario estaar en el directorio
tarea01/spam.py SMSSpams.txt

Debe de estar el archivo de texto en la misma carpeta para que funcione, y para ejecutarlo una vez con todas las bibliotecas instaladas, ejecutas:
	python3 spam.py
